# TransferWithEther

TransferWithEther 是一個簡單的圖形介面程式，協助兩台電腦透過網路線快速傳送檔案。程式同時提供發送端與接收端模式，可檢查連線狀態、選擇檔案並顯示傳輸進度。

## 功能

- **模式切換**：發送端與接收端在同一個介面中切換。
- **連線檢查**：發送端可在傳送前確認遠端是否可以連線。
- **檔案選擇**：使用檔案選擇器挑選要傳送的檔案。
- **開始/取消控制**：開始傳輸或隨時取消目前的操作。
- **進度顯示**：使用進度條與狀態訊息呈現傳輸進度與狀態。

## 系統需求

- Python 3.9 或更新版本
- 同網段下兩台電腦，可直接透過網路線或區域網路連線

## 安裝與執行

1. 建立虛擬環境（建議）並安裝 Python 依賴：

   ```bash
   python -m venv .venv
   source .venv/bin/activate  # Windows 請改用 .venv\Scripts\activate
   python -m pip install --upgrade pip
   ```

2. 啟動圖形介面：

   ```bash
   python main.py
   ```

## 使用流程

1. 在接收端電腦：
   - 啟動程式後，選擇「Receiver」模式。
   - 選擇要儲存檔案的資料夾。
   - 輸入要監聽的連接埠（預設 5001），按下 **Start** 開始等待。

2. 在發送端電腦：
   - 啟動程式後，保持在「Sender」模式。
   - 填入接收端的 IP 位址與對應的連接埠。
   - 按下 **Check Connection** 確認網路是否暢通。
   - 選擇要傳送的檔案，按下 **Start** 開始傳輸。

3. 傳輸過程中可透過 **Cancel** 按鈕終止操作，程式會顯示目前進度與狀態資訊。

## 注意事項

- 兩端需允許指定連接埠的進出連線，必要時調整防火牆設定。
- 若要以直連網路線傳輸，請確認兩台電腦已設定對應的 IP 位址。
- 目前一次僅支援單一檔案的傳輸。
